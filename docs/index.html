About WallBreaker

WallBreaker is a Telegram-based mini app focused on gameplay, ranks, ad-based progression, draw participation and prize pool competition.

The project is built as a Telegram Mini App frontend on GitHub Pages, while the secure game backend is separated and handles user auth, balance logic, taps, ranks, referrals, draws and purchases.
